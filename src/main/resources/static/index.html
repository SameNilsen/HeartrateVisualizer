Visualization of
80/20 rule for cardio


This application aims to show and visualize how much time is spent in each heartrate zone. This will be useful if you seek to follow the 80/20 rule in endurance/cardio training. The application collects your data from Strava, so it is required to give authorization for it to be able to access your activities from the Strava API. The authentication is done by Strava using OAuth2, and no personal data will be stored by the application.


Please press Authorize to get started
